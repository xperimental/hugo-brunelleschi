<h2>README</h2>

Developed by Kit MacAllister, this theme is based on the WordPress Team's Twenty Ten theme
with implementation of HTML5 structure and the <a href="http://html5boilerplate.com/">HTML5 Boilerplate</a>.

This theme also uses the <a href="http://cssgrid.net/">1140 CSS grid</a>, which allows it's flexible structure.
This theme provides a sidebar at high resolutions which becomes a footer at lower resolutions.

To contact the author email <a href="mailto:kit.macallister@gmail.com" target="_blank">kit.macallister@gmail.com</a>.
